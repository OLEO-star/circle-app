# 9×4 新4学部 — 6大学帯シラバス採取 × 22軸 充足性検証（2026-06-18）

## 目的（かずきの問い・2026-06-18）
9×4（36学部）に強く前向きだが確定の前提として「**22軸より軸が増えるなら、追加質問がどの程度になるか**」を見積もる。
→ 新4学部（**歯学・土木都市環境工・経営工・材料工**／栄養は落とす）について、過去制作と同じく**6大学帯のシラバスからデータを採取**し、現22軸にスコアを当てはめ、既存32学科と被らないか・被るなら何軸足せば良いかを判定する。

## 方法
1. 4学部それぞれ、6偏差値帯から実在大学を1校ずつ選び、カリキュラム/必修/実習/資格/進路を採取（リサーチ4並列）。
2. 採取に基づき22軸スコアを確定（**類推値→実データ値**に更新）。
3. v2判定式（36集合・mixed=全22軸計測・γ=0.7・ゲート）で、各学部の「理想だがブレた受験生」を多数生成し top1復元率を測定（σ=0.10/0.15）。
4. 各新学部の最近傍twinと、現22軸のどの軸で分離しているかを確認。
   - 再現コード: `src/lib/recat_axis_check.ts`（採取値を反映済み）。

参考: 既存の最も近いペアも同水準でズレている — 数学↔物理 90%、商↔経営 89%、化学↔応用化学 91%（σ=0.15）。

## 結論：**必須の新軸＝0本。9×4は現22軸で成立する。**
- 新4学部はいずれも**現22軸で一意に立つ**（σ=0.10で全て95%以上、うち3学部は99-100%）。分離はすべて既存軸（MAKE/FIELD/CERT/PROC/BIZ/CODE/MATH/ABS）の組合せで表現できる。
- 「土木vs建築・経営工vs経営・材料vs化工は現22軸で分離しきれない恐れ」という当初の懸念は、**データで否定された**（3学部は完璧、材料工も化工との混同は解消）。
- **材料工だけがやや弱い（σ=0.15で85%）**。ただし真の双子は化工ではなく**電気電子**で、混同率13%は既存の 数学↔物理 と同程度＝**現システムが既に許容している水準**。分野破綻ではない。

### 追加質問の見積もり（かずきの前提条件への直接回答）
- **軸は増えない＝0本** → 厳密には**追加質問0問でも9×4は成立する**。
- ただし新4学部の分離は **PROC軸（量産・工程）が要石**（土木=高/経営工=高/材料=低 で三者を割る）。その PROC が現在**1問しかない**（BIO=2問）＝分離の主役が最も薄いという逆転状態（`2026-06-15-axis-assignment-audit.md`既出）。
- **頑健性を担保するなら推奨＝+2〜3問**：PROC 1→3（+2）、任意で BIO 2→3（+1）。**新軸ではなく既存薄軸の増強**。sciences/mixed版に追加。
- 新軸を立てるとしても候補は **SOLID（固体物性志向）1本**（材料0.9/化工0.2/物理0.7/電気電子0.5）で材料↔電気電子↔化学系を一段クリアにできるが、=+3問・全32学科の再採点コストを伴い、現時点では**過剰（"必要"でなく"あれば綺麗"）**。

## 9×4 GO判断への含意
質問数のコスト面では 9×4 は**ほぼ無料**（必須+0問、推奨でも+2〜3問）。当初「9×4＝診断システムの大改修」と見ていた(b)質問・軸の追加リスクは、**実データ採取により小さい**ことが判明。残る9×4のコストは (a)新4学部のdepartments追加（本採取値で対応可）と (c)カテゴリ9化（リング描画）のみ。
